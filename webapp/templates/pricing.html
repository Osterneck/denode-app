Frequently Asked Questions

What databases does DEnode support?

PostgreSQL, MySQL, SQLite, and Microsoft SQL Server — anything SQLAlchemy can connect to via a URL.

Do you store my database credentials?

No. Connection URLs are used only during the extraction request and are never persisted. Only the extracted schema metadata is stored.

Can I cancel anytime?

Yes. Cancel from the billing portal at any time. You keep Pro access until the end of your billing period.

What does "schema analysis" mean on the free plan?

One full analysis cycle: schema extraction + query log analysis + recommendations + SQL generation for one database. Upgrade for unlimited databases.
